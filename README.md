# 🐍 Snake Animation Setup

The snake animation is automatically generated by GitHub Actions workflow.

**Workflow file:** `.github/workflows/snake.yml`

---

# ⚙️ Important Configuration

To ensure the snake animation works properly, follow these steps:

1. Go to your repository settings:
   ```text
   Repository Settings
   → Actions
   → General
   → Workflow Permissions
   ```

2. Enable:
   ```text
   ✅ Read and Write Permissions
   ```

3. The workflow will automatically run every 12 hours and whenever you manually trigger it.
